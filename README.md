# 🎓 Student Placement Tracker

A modern web-based Student Placement Tracker developed using HTML, CSS, JavaScript, Firebase Firestore, Chart.js, and jsPDF. This application helps colleges and placement coordinators efficiently manage student placement records, generate reports, and analyze placement statistics through interactive dashboards.

---

## 🚀 Features

### 🔐 Authentication

* Secure Login System
* Session-based Access Control

### 👨‍🎓 Student Management

* Add Student Records
* Edit Student Details
* Delete Student Records
* Upload Student Photos
* Search Students Instantly
* Filter Students by Company

### 🏢 Company Reports

* Company-wise Placement Statistics
* Applied Students Count
* Selected Students Count
* Rejected Students Count

### 📊 Analytics Dashboard

* Total Students Overview
* Selected Students Count
* Rejected Students Count
* Placement Percentage Calculation
* Pie Chart Visualization
* Bar Chart Visualization
* Recent Placements Table

### 📄 Report Generation

* PDF Export
* CSV Export

### ⚙️ Additional Features

* Dark Mode
* Responsive Design
* Firebase Firestore Integration
